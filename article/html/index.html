Не останавливайтесь на одной статье, ресурсе, так как сколько людей столько и мнений, поэтому подход к предоставлению информации может отличаться.

Введение в html

После того как вы скачали текстовый редактор, создайте файл index.html обязательно в таком расширении (html). Поместите в созданный файл код что я указал ниже и далее запустите файл через любой браузер. Вы должны увидеть надпись "Hello World!". Hello World традиционно используется в качестве первой программ для изучения любого языка, не обязательно должна быть именно эта фраза, можете написать все что вам хочется. Попробуйте заменить этот текст и перезапустить страницу в браузере, вы должны увидеть имзенения. Код ниже - это стандартная основа для html разметки, лучше всего придерживаться ее, но если вы удалите часть тегов html можно заметить что браузер все равно отработает нормально.

<!DOCTYPE html>
<html lang="ru">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Document</title>
</head>
<body>
  <h1>Hello World!</h1>
</body>
</html>

Давайте разберем что написано в html коде. <!DOCTYPE html> для начала достаточно знать что это обозначает тип документа. <html lang="ru"></html> является контейнером в котором находится содержимоей всей вашей страницы сайта. Обратите внимание, что тег html состоит из двух частей <html lang="ru"> и </html> так обозначаются теги вида контейнеров, открывающий и закрывающий тег.

Внутри подобные теги, как правило содержат либо другие теги либо текст. Большинство тегов html как правило имею такойже вид, но есть и исключения. далее идет контейнер <head></head> данный тег служит помощью для браузера, чаще всего тут указываются данные как должна отображаться страница, например кодировка текста, <meta charset="UTF-8"> информация как должна отображаться страница в мобильном устройстве <meta name="viewport" content="width=device-width, initial-scale=1.0"> следующим на очереди идет title <title>Document</title> - это заголовок вашей страницы, наведя на вкладку браузера можно увидеть этот текст, это достаточно важный тег и не стоит его игнорировать.

Например для "СЕО" продвижения, в кратце поисковые системы используют этот тег для отображения заголовка в своем поиске или определении подходит ли ваша страница по поисковому запросу. Далее тег head закрывается и открывается следующий тег <body></body> - в этом теге находится все что будет отображаться на вашей странице. То есть тело документа html. В нашем случае, это <h1>Hello World!</h1> так же можно увидеть тег <h1></h1> это тег заголовка. Заголовки суещствуют 6 видов, 1 верхнего уорвня и по лестнице до самого низкого уровня 6.

Существует огромное можество тегов, каждый отвечает за свою часть на сайте. Лучше всего перед тем как продолжить обучение и переходить уже к css изучить как можно больше тегов и из свойства. Например тег a <a href="url">Текст ссылки</a> это тег ссылки на другой документ / веб страницу. тег <img src="url" alt=""> это тег отвечает за отображение картинок (изображений) то что я указал как "url" - это путь до документа на вашем или другом сервере. Вы можете подтягивать на свой документ изображения с других сайтов если это необходимо. Вот например. <img src="https://hotgeo.ru/uploads/posts/2017-10/1509441205_gctv-google.jpg" alt="Изображение офиса Google" style="max-width:300px"> и как это отображается в веб странице:

Для большего понимания, чаще используйте консоль разработчика для изучения других сайтов, например этого, он достаточно просто сверстан, чтобы можно было разобраться что тут происходит. Позднее когда я начну говорить о CSS чуть подробнее раскажу как на этом сайте устроен внешний вид.

Как можно понять из описанного ранее, у тегов есть атрибуты, для тега img использовался атрибут src,alt и style.

  • Атрибут style относится к css стилизации тега.
  • src - путь по которому будет браузер искать изображение
  • alt описание изображения

Некоторые атрибуты, значительно изменяют внешний вид и смысл использования тега, понимание этого поможет в работе с html. Приведем еще пример тега input

<input type="text">
<input type="checkbox">
<input type="range">

Атрибут type для тега input

  • свойство "текст"
  • свойство "чекбокс"
  • свойство "рендж"

Для понимания, тег "input" часто используется в формах, например регистрация, авторизация, опросы и тд. например ваш пол М Ж
вот как влияют атрибуты на свойства html тегов. Это только пример, на самом деле у одного тега input куда больше вариантов одного только атрибута type.
